============
Installation
============


Stable release
--------------

To install hi, run this command in your terminal:

.. code-block:: console

    $ pip install hi

This is the preferred method to install hi, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From source
-----------

The source for hi can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/Dragonlord1005/hi

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/Dragonlord1005/hi/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ pip install .

.. _Github repo: https://github.com/Dragonlord1005/hi
.. _tarball: https://github.com/Dragonlord1005/hi/tarball/master
